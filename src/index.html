Esta es un proyecto para probar cómo se debe enlazar un formulario con el que se capturan datos en una página web y estos se envían para almacenarlos en una base de datos.

En este campo puede incluir cualquier tipo de dato ya que no se realiza validación de ningún tipo; se deja a su discresión si desea/necesita incluir mayúsculas, minúsculas, números, caracteres especiales, etc.
